# 隐私政策 (Privacy Policy)

**生效日期：2026年1月14日**
**更新日期：2026年1月19日**

欢迎使用 **GLaDOS 自动签到助手** (GLaDOS Checkin Helper)。我们非常重视您的隐私。本政策旨在说明本扩展程序如何处理数据。

## 1. 数据收集与使用
本扩展程序是一个纯本地运行的自动化工具。
* **个人信息**：本扩展**不会**收集、上传或存储您的任何个人身份信息（如姓名、电子邮件、密码等）。
* **登录凭证**：本扩展利用您浏览器中现有的登录状态（Cookie）与 `glados.cloud` 进行通信。本扩展本身**不会**接触、读取或存储您的登录密码。

## 2. 数据存储
所有相关配置和历史记录均存储在您的本地计算机上：
* **本地存储 (Chrome Storage)**：用于记录您的签到时间设置和最近 30 次的签到结果日志。
* **数据清理**：如果您卸载本扩展程序，存储在本地的所有相关数据都将被 Chrome 浏览器自动清除。

## 3. 第三方共享
* **数据不外流**：本扩展**不会**将您的任何数据共享、出售或传输给任何第三方机构或个人。
* **外部通信**：本扩展仅与 `https://glados.cloud` 官方 API 进行必要的数据交换以完成签到动作。

## 4. 权限说明
本扩展申请的权限仅用于实现核心功能：
* `storage`: 用于保存您的个人设置和签到历史。
* `alarms`: 用于设定定时检查任务，确保自动化执行。
* `notifications`: 用于在签到任务完成后向您发送系统反馈。
* `host_permissions`: 仅用于访问 GLaDOS 官方页面进行签到操作。
* `declarativeNetRequest`：用于在签到请求中安全地修改 Origin 和 Referer 头部，以确保请求符合官方服务器的安全校验逻辑，防止因身份识别问题导致签到失败。

## 5. 联系方式
如果您对本隐私政策有任何疑问，可以通过 GitHub 提交 Issue 与我联系。
